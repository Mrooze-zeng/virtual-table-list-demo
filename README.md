# table-demo

[预览地址](https://mrooze-zeng.github.io/virtual-table-list-demo/)

## 说明

目前本 demo 采用的是多 table 叠加的方式达到显示效果，未采用`position:sticky`的方式,采用`position:sticky`的方式应该会减少大量的 dom 生成

